momPaginator Example

by Mike Evans

The momPaginator turns your ordinary HTML table into an advanced datatable capable of retrieving data from an external ReST-based API, paginating the data, moving through the pages of data, and even sorting the data on individual columns.

This example shows a simple query of all GitHub users using the GitHub ReST API.
